# pet-sitter-log

Serverless backend for a pet-sitter daily-care log: a sitter records walks, meals, and notes; the owner follows along through a private share link, no account needed.

Built with AWS CDK (TypeScript): DynamoDB single-table, Lambda (Node 20), API Gateway HTTP API, deployed to `ap-southeast-1`.
